# Blueprint MCP Agent Rules

Blueprint MCP stores local architecture memory in `.blueprint/`.

Before broad codebase exploration:

1. Read `.blueprint/brief.md` when it exists.
2. Search `.blueprint/groups/*.md` with task-specific keywords.
3. Read only the smallest useful set of group docs.
4. Inspect source code only where the docs are insufficient, stale, or edits are needed.

Blueprint files are orientation, not authority. Source code remains the source of truth.
